# Contributing

Thanks for taking interest in improving Prompt Optimization Engine.

## Goals
- Provide a free and transparent prompt-optimization toolkit.
- Offer repeatable, testable optimization strategies.
- Make it easy to deploy and extend.

## Getting Started
1. Fork and clone the repo.
2. Create a virtual environment.
3. Install dependencies from `backend/requirements.txt`.
4. Run tests with `pytest`.

## Good First Issues
- Add more templates for common use cases.
- Add a new scoring heuristic and tests.
- Improve UI presets in `app.js`.
- Add CSV import/export to the frontend.
- Improve documentation for deployment.
- Add example prompts to `README.md`.

## PR Checklist
- [ ] Tests added/updated.
- [ ] Linting passes.
- [ ] Changes documented.
- [ ] No breaking changes without a note.
